2、安装编辑器

Git Book legacy（旧版）官网下载GitBook Editor（目前好像官网不提供离线编辑器了，点击链接跳转到了新版官网），执行安装文件。

打开Editor，发现是要登录的。

忙活好久，去官网注册账号，发现legacy（旧版）官网和新版官网的账号不是通用的！！

旧版官网注册不成功，新版注册成功，但是不能在Editor上登录成功，桌面编辑器貌似只针对旧版账号可用！

失望之至的时候，看到Editor欢迎页有“Do that later”小字T_T

点击“Do that later”，可以进入编辑器！
